Interactive DOM FAQ Explorer

Element Section

'document.querySelector()' returns the first element within the document that matches the specified CSS selector.
'document.querySelectorAll()' returns a static NodeList of all elements that match the specified CSS selector.
'document.getElementById()' returns the element with the specified ID attribute value.
'document.getElementsByClassName()' returns a live HTML Collection of all elements with the specified class name.
'document.getElementsByTagName()' returns a live HTML Collection of all elements with the specified tag name.

Element Creation & Insertion

'document.createElement()' creates a new HTML element specified by tagName.
'element.appendChild()' adds a node to the end of the list of children of a specified parent node.
'element.removeChild()' removes a child node from the DOM and returns the removed node.
'element.replaceChild()' replaces a child node within a specified parent node with a new node.
'document.createTextNode()' creates a new text node that can be inserted into the DOM using methods like 'appendChild()'.
'element.insertBefore()' inserts a new child node before a specified existing child node of the parent.

Element Modification

'element.innerHTML' gets or sets the HTML content of an element, allowing you to modify the content dynamically.
'element.textContent' gets or sets the text content of an element and its descendants, providing a way to read or update the textual content.
'element.remove()' removes the element from the DOM directly, without needing to call removeChild() on its parent.

Attributes and Classes

'element.setAttribute()' adds a new attribute or changes the value of an existing attribute on the specified element.
'element.getAttribute()' returns the value of a specified attribute on the element.
'element.removeAttribute()' removes an attribute from the specified element.
'element.classList.add()' adds one or more class names to an element.
'element.classList.remove()' removes one or more class names from an element.
'element.classList.toggle()' adds a class if it is not present and removes it if it is.
'element.classList.contains()' checks if an element has a specified class and returns a boolean value.
'element.hasAttribute()' returns a boolean indicating whether the element has the specified attribute.

DOM Traversal

'element.children' returns only element nodes, while 'element.childNodes' includes all child nodes (including text and comment nodes).
Both 'element.parentNode' and 'element.parentElement' return the parent of the specified element. 'element.parentElement' specifically returns only an element node.
'element.firstChild' can return any node (including text), while 'element.firstElementChild' returns only the first child element.
'element.nextSibling' returns the next node, while 'element.nextElementSibling' returns the next element sibling.
'element.lastElementChild' returns the last child element of a specified element. It ignores text nodes and comment nodes, returning only HTML elements.
'element.lastChild' returns the last child node of a specified element. This can be an element node, a text node, or even a comment nodeâ€”not just elements.

Events

'element.addEventListener()' attaches an event handler to a specified element, enabling the element to respond to the event.

Miscellaneous / Utilities

'element.style' gets or sets the inline CSS styles of an element.
'document.body' returns the body element of the current HTML document.
